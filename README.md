# Personal Portfolio Website

A minimalist personal portfolio website with social media links.

## Features

- Clean, minimalist design
- Responsive layout
- Dark mode support (automatic based on system preference)
- Social media links (Twitter, GitHub, LinkedIn, Email)
- Project showcase section

## Customization

1. Replace "Your Name" with your actual name in `index.html`
2. Update the tagline to match your profession/description
3. Edit the about section with your own bio
4. Add your actual projects to the project cards
5. Update all social media links with your profiles:
   - Twitter: Replace `yourusername` with your Twitter handle
   - GitHub: Replace `yourusername` with your GitHub username
   - LinkedIn: Replace `yourusername` with your LinkedIn username
   - Email: Replace `your.email@example.com` with your email

## Running Locally

Simply open `index.html` in your web browser.

## Deployment

You can deploy this website to:

- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Just upload the files and you're done!
